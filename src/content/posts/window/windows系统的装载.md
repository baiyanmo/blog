# windows系统的安装

win11系统的下载网址为

[下载 Windows 11](https://www.microsoft.com/zh-cn/software-download/windows11)

win10的下载的地址为

[下载 Windows 10](https://www.microsoft.com/zh-cn/software-download/windows10)

找到此选项

![Windows系统安装选项](./images/1.png)

下载完成后点击exe文件

![exe文件](./images/2.png)

选择接受![选择接受](./images/3.png)

选择u盘为介质，iso为镜像文件一般下载在虚拟机中

![选择介质](./images/4.png)

选择要下载的u盘

![选择U盘](./images/5.png)

![制作进度](./images/6.png)

待进度完成则系统u盘制作完成
