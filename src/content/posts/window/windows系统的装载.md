# windows系统的安装

win11系统的下载网址为

[下载 Windows 11](https://www.microsoft.com/zh-cn/software-download/windows11)

win10的下载的地址为

[下载 Windows 10](https://www.microsoft.com/zh-cn/software-download/windows10)

找到此选项

![Windows系统安装选项](./img/step1.png)

下载完成后点击exe文件

![exe文件](./img/step2.png)

选择接受![选择接受](./img/step3.png)

选择u盘为介质，iso为镜像文件一般下载在虚拟机中

![选择介质](./img/step4.png)

选择要下载的u盘

![选择U盘](./img/step5.png)

![制作进度](./img/step6.png)

待进度完成则系统u盘制作完成
